# IGF V1.0

## IGF Gathers information about a target and its environment.

DNS LOOKUP<br/>
Subdomains<br/>
Directories and Filenames<br/>
Ports and Services<br/>
HTTP requests<br/>
Whois information<br/>
Domain to IP converter<br/>

More to come in the next release.
